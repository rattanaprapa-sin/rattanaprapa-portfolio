# Personal Portfolio

โปรเจกต์เว็บพอร์ตโฟลิโอส่วนตัว (Personal Portfolio) พัฒนาด้วย React และ Express.js

## 🛠 Tech Stack
- **Frontend:** React (Vite)
- **Backend:** Express.js (Node.js)

## 📁 โครงสร้างโปรเจกต์
- `/frontend` - โค้ดส่วนหน้าบ้าน (UI/UX)
- `/backend` - โค้ดส่วนหลังบ้าน (API & Server)

## 🚀 วิธีการติดตั้งและรันโปรเจกต์ (Local Development)

1. **ติดตั้ง Dependencies:**
   รันคำสั่งนี้ที่โฟลเดอร์หลัก (Root) เพื่อติดตั้ง Package ทั้งหมดสำหรับทั้งโปรเจกต์หลัก, frontend และ backend
   ```bash
   npm run install-all
   ```

2. **รันโปรเจกต์:**
   รันคำสั่งนี้ที่โฟลเดอร์หลัก เพื่อเปิดใช้งานทั้ง Frontend และ Backend ไปพร้อมๆ กัน
   ```bash
   npm start
   ```
