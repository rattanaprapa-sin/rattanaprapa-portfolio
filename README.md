# 👨‍💻 Rattanaprapa Sinkrathok - Personal Portfolio

ยินดีต้อนรับสู่เว็บพอร์ตโฟลิโอ พื้นที่นี้ถูกจัดทำขึ้นเพื่อรวบรวมผลงาน ประสบการณ์การทำงาน และทักษะทางด้านซอฟต์แวร์ (Software Development) ปัจจุบันโปรเจกต์นี้โฟกัสการพัฒนาในส่วนของ **Frontend (React)** เป็นหลัก โดยได้เตรียมโครงสร้างแบบ Monorepo เพื่อรองรับการพัฒนา API และฟีเจอร์ฝั่ง Backend เพิ่มเติมในอนาคต

## 🚀 Live Demo
- **URL:** [https://rattanaprapa-portfolio.vercel.app/](https://rattanaprapa-portfolio.vercel.app/)

## 🛠 Tech Stack (เทคโนโลยีที่ใช้สร้างเว็บนี้)
- **Frontend:** React (Vite), HTML5, CSS3, Bootstrap 5
- **Tools:** Git, VS Code
- *(Planned)* **Backend:** Express.js (Node.js)

## 🌟 ผลงานที่โดดเด่น (Featured Projects ใน Portfolio)
1. **Candidate Pool Management System** - ระบบจัดการรายชื่อผู้สมัคร แบบ Full-stack (React, Express.js, SQL Server)
2. **Concert Ticket Booking Website** - แพลตฟอร์มจองตั๋วคอนเสิร์ตพร้อมระบบเลือกที่นั่งแบบอินเทอร์แอคทีฟ (Python, Django, JS)
3. **Movie Ticket Booking Website** - ระบบจองตั๋วหนังและเลือกรอบฉาย (Python, Flask, JS)
4. **Image Guessing Mobile Game** - เกมทายภาพปริศนาบนมือถือ (Thunkable)

## 📁 โครงสร้างโปรเจกต์
โปรเจกต์นี้มีการวางโครงสร้าง (Monorepo) เพื่อเตรียมสเกลเป็น Full-Stack ในอนาคต โดยแบ่งเป็น:
- `/frontend` - โค้ดส่วนหน้าบ้าน (UI/UX)
- `/backend` - โค้ดส่วนหลังบ้าน (เตรียมพร้อมสำหรับการเขียน API ในอนาคต)

## ⚙️ วิธีการติดตั้งและรันโปรเจกต์ (Local Development)

1. **Clone Repository:**
   ```bash
   git clone https://github.com/rattanaprapa-sin/rattanaprapa-portfolio.git
   cd rattanaprapa-portfolio
   ```

2. **ติดตั้ง Dependencies:**
   ```bash
   cd frontend
   npm install
   ```

3. **รันเซิร์ฟเวอร์ (Frontend):**
   ```bash
   npm run dev
   ```
